⚙️ Conectar con Google Sheets

Introduce la URL de tu Google Apps Script para sincronizar los datos automáticamente.

Pasos para obtener la URL:
1. Google Sheet → Extensiones → Apps Script
2. Pega el código de AppsScript_CuentasRojas.js y guarda
3. Implementar → Nueva implementación
4. Tipo: Aplicación web · Ejecutar como: Tú
5. Acceso: Cualquier usuario (anónimo, sin Google)
6. Copia la URL /exec y pégala aquí
